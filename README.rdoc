= Next-L Enju Leaf公共図書館拡張版

Next-L Enju Leaf は、{Project Next-L}[http://www.next-l.jp] で開発している図書館管理システムです。
公共図書館拡張版は、株式会社三菱総合研究所と株式会社ミライト情報システムによって、公共図書館向けに機能強化したものです。

Next-L Enju Leaf is an integrated library system developed by {Project Next-L}[http://www.next-l.jp]. Public Library Extension is developed by Mitsubishi Research Institute, Inc. and Open Technologies Corporation.

== Project Next-L とは (What is Project Next-L?)
{Project Next-L}[http://www.next-l.jp] とは、日本の図書館関係者有志の手で新しい図書館管理システムを作り上げる
プロジェクトです。株式会社三菱総合研究所と株式会社ミライト情報システムは本プロジェクトを支援します。

{Project Next-L}[http://www.next-l.jp] is a project to build a new integrated library system maintained 
by Japanese volunteers interested in libraries. Mitsubishi Research Institute, Inc. 
and Mirait Information Systems Corporation endorses this project.
